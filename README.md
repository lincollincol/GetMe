## Download GetMe

### Gradle
``` groovy
allprojects {
  repositories {
    maven { url 'https://jitpack.io' }
  }
}

dependencies {
  implementation 'com.github.lincollincol:GetMe:1.0'
}

  ```
### Maven
```xml
<repositories>
  <repository>
    <id>jitpack.io</id>
    <url>https://jitpack.io</url>
  </repository>
</repositories>

<dependency>
  <groupId>com.github.lincollincol</groupId>
  <artifactId>GetMe</artifactId>
  <version>1.0</version>
</dependency>
```
